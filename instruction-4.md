硬件控制与系统命令

### `force` — 继电器强制接管

绕过故障保护和逻辑状态机，人工强行闭合/断开继电器。

| 用法 | 说明 |
|------|------|
| `force auto` | 交回控制权给 App_Logic 状态机 |
| `force override <Pos> <Neg> <Pre> <Heat> <QF> <Limit>` | 强控指令，1 为吸合，0 为断开。例: `force override 1 1 0 0 1 0`（强行闭合主正、主负和断路器，断开预充、加热和限流） |

### `sys` — 系统级复位

| 用法 | 说明 |
|------|------|
| `sys reset fault` | 向仲裁器和状态机发送故障清除指令（常用于 10 次死锁后的强制复归） |
| `sys reset param` | 将 KVDB 系统配置参数恢复出厂设置（需手动重启生效） |
| `sys reset mcu` | 触发看门狗或 NVIC，进行系统级热重启 |
