BMS CLI命令速查汇总表
| 指令 (英文) | 功能说明 (中文) |
|-------------|-----------------|
| `help` | 查看所有命令列表 |
| `info` | 打印当前系统"体检报告"(黑板状态) |
| `task` | 打印 RTOS 任务调度与 CPU/堆栈负载 |
| `fault` | 详细诊断并打印当前激活的故障等级 |
| `sim` | 开启电池模型仿真 (HIL测试) |
| `inj` | 采集数据强制物理偏移 (模拟故障) |
| `bal` | 强控硬件均衡通道 (测试发热/驱动) |
| `calib` | 强制标定底层 EKF 估算矩阵与SOC |
| `est` | 打印或格式化 KVDB 中的估算参数 |
| `force` | 强行接管/断开主回路继电器 |
| `sys` | 系统复位 (故障复归/恢复出厂/重启) |
| `get` | 读取系统阈值或配置参数 |
| `set` | 修改并保存系统阈值或配置参数 |
| `loglevel` | 更改控制台日志打印级别 (0~3) |
| `log` | 读取或清除 Flash 黑匣子故障快照 |
