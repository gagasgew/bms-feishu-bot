参数读写与日志配置

### `get` — 读取配置表参数

| 用法 | 说明 |
|------|------|
| `get <param_name>` | 读取指定参数当前值，根据类型（Threshold 阈值型、U32、I32）自动解析。阈值型会同时打印其保护、延时和恢复条件（I, II, III, RST） |
| `get all` | 读取所有配置参数 |

### `set` — 修改配置表参数

修改后立即生效于 RAM，并异步派发给 DrvTaskDB 存入 Flash，掉电不丢失。

| 用法 | 说明 |
|------|------|
| `set <param_name> <value>` | 单一数值型参数写入 |
| `set <param_name> <I级> <II级> <III级> <RST恢复值>` | 多级阈值型参数写入 |

### `loglevel` — 控制台输出过滤

| 用法 | 说明 |
|------|------|
| `loglevel <0-3>` | 调整全局 Log 打印的最低显示级别。0 = DEBUG, 1 = INFO, 2 = WARN, 3 = ERROR |

### `log` — 黑匣子数据追溯

| 用法 | 说明 |
|------|------|
| `log read <N>` | 从时序数据库提取并打印最近 N 条故障快照（包含当时的温度、电压、电流断面） |
| `log clear` | 抹除内部 Flash 中的所有黑匣子快照 |
